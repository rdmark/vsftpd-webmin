Manual configuration
This page allows you to manage your vsftpd.conf file. Each option is explained in further detail in the man page of vsftpd.

You can select from one the following actions:

  • Save changes: write the changes to the vsftpd.conf file
  • Undo unsaved changes: reload the vsftpd.conf configuration without saving current changes
  • Rollback to previous version: before changes are saved a backup is made of the vsftpd.conf file, this option allows you to restore the backup
